OverView Download et Installation Démonstration

Généralitées

Pour toutes les démonstrations nous avons un fichier index.html qui contient le site de façons générale, Ping_Pong.js qui est la librairie, data.json qui est le fichier de données, template.json qui est le template.

Exemple 1 : Chargement d'un template

index.html

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Title of the document</title>
</head>

<body>
<button onclick="calltemplate()">Load</button>
<div template-id="routehtml1"></div>

</body>
<script src="Ping_Pong.js"></script>
<script>
function calltemplate(){
PingPong.render('routehtml1', 'template.html', {})
}
</script>
</html>
