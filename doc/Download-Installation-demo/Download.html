OverView Download et Installation Démonstration

Téléchargement

Vous avez 2 méthode pour télécharger la librairie.

  • Télécharger le fichier ping_pong.js du Repository
  • Télécharger ou clonner le Repository git clone https://github.com/yolocodefrench/ping_pong.js.git

Installation

Pour installer la librairie il vous faudra ajouter le fichier ping_pong.js dans un dossier de votre projet, plus l'ajouter à votre HTML.

  • <script type="text/javascript" src="ping_pong.js" ></script>

Enfin, si vous souhaitez utiliser la fonctionnalité statique, il vous fautra créer le fichier routage.json à la racine de votre document html.

  • {
    "routes": [
    {"HTML_ID": "routehtml1", "Template_File": "templateFile.html", "Data_File": "dataFile.php" },
    {"HTML_ID": "routehtml2", "Template_File": "templateFile2.html", "Data_File": "dataFile1.php" }
    ]
    }
