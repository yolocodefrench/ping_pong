pp_createFormData

Permet d'obtenir un objet FormData pour les appels de site à partir d'un objet.

pp_createFormData(object)

Arg object Object L'objet à convertir.
Return FormData | null FormData L'objet FormData obtenu à partir de l'objet ou null si il y a une erreur.
